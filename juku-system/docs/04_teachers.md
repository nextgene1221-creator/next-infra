# 講師管理 詳細仕様書

## 基本情報

| ページ | パス | ソースファイル | レンダリング | アクセス可能ロール |
|--------|------|---------------|-------------|-------------------|
| 講師一覧 | `/teachers` | `src/app/(authenticated)/teachers/page.tsx` | サーバーサイド | admin |
| 講師詳細 | `/teachers/:id` | `src/app/(authenticated)/teachers/[id]/page.tsx` | サーバーサイド | admin |
| 講師登録・編集 | `/teachers/:id/edit` | `src/app/(authenticated)/teachers/[id]/edit/page.tsx` | クライアントサイド | admin |

### API エンドポイント

| メソッド | パス | 説明 | 権限 |
|---------|------|------|------|
| GET | `/api/teachers/:id` | 講師情報取得 | 認証済み全ロール |
| POST | `/api/teachers` | 講師新規登録 | admin のみ |
| PUT | `/api/teachers/:id` | 講師情報更新 | admin のみ |
| GET | `/api/teachers-list` | 講師名リスト（シフト登録用） | admin のみ |

## 1. 講師一覧ページ (`/teachers`)

### 機能説明

#### 検索・フィルタ機能

| パラメータ | 説明 | 選択肢 |
|-----------|------|--------|
| `q` | 名前での部分一致検索 | 自由入力テキスト |
| `status` | ステータスフィルタ | 全ステータス / 稼働中(active) / 非稼働(inactive) |

#### 新規登録ボタン
- ヘッダー右側に常時表示（adminのみがアクセス可能なため）
- 遷移先: `/teachers/new/edit`

### デザイン仕様

#### テーブルカラム

| カラム | 表示内容 | スタイル |
|--------|---------|---------|
| 名前 | `user.name` | 青色リンク → 講師詳細ページ |
| 担当科目 | `subjects`（JSON配列をカンマ区切り表示） | テキスト |
| 雇用形態 | full_time → 「常勤」/ part_time → 「非常勤」 | テキスト |
| 担当生徒数 | `student_teacher` の紐付け件数 | テキスト |
| ステータス | バッジ表示 | 色分けバッジ |

#### ステータスバッジの色分け

| ステータス | 表示名 | バッジ色 |
|-----------|--------|---------|
| active | 稼働中 | 緑（`bg-green-100 text-green-800`） |
| inactive | 非稼働 | 赤（`bg-red-100 text-red-800`） |

#### レイアウト
- 検索フォーム: 生徒一覧と同一デザインパターン（白背景カード、横並びフォーム要素）
- テーブル: 生徒一覧と同一デザインパターン
- 0件時: 「講師が見つかりません」を中央表示

## 2. 講師詳細ページ (`/teachers/:id`)

### 機能説明

3つのセクションで講師情報を表示。

#### セクション構成

| セクション | 表示内容 | グリッド位置 |
|-----------|---------|-------------|
| 基本情報 | メール、電話番号、担当科目、雇用形態、ステータス | 左列 |
| 担当生徒 | 紐付いている生徒名と担当科目 | 右列 |
| 最近のシフト | 直近10件のシフト（日付、時間、ステータス、備考） | 全幅（`lg:col-span-2`） |

#### 編集ボタン
- ヘッダー右側に「編集」ボタン → `/teachers/:id/edit`

### デザイン仕様

#### 基本情報セクション
- 定義リスト形式（ラベル幅 `w-32`）
- 表示項目: メール、電話番号、担当科目（カンマ区切り）、雇用形態、ステータス

#### 担当生徒セクション
- リスト形式、各項目 `bg-gray-50 rounded p-2`
- 生徒名: 青色リンク → 生徒詳細ページ
- 科目: 右寄せテキスト

#### シフトテーブル
- 全幅カード（`lg:col-span-2`）
- カラム: 日付、時間（開始-終了）、ステータスバッジ、備考

#### シフトステータスバッジの色分け

| ステータス | 表示名 | バッジ色 |
|-----------|--------|---------|
| scheduled | 予定 | 青（`bg-blue-100 text-blue-800`） |
| confirmed | 確定 | 緑（`bg-green-100 text-green-800`） |
| cancelled | キャンセル | 赤（`bg-red-100 text-red-800`） |

## 3. 講師登録・編集ページ (`/teachers/:id/edit`)

### 機能説明

#### 新規登録モード
- パスワード入力フィールド表示（必須）
- POST `/api/teachers` で新規作成
- 成功時 → 講師詳細ページへ遷移

#### 編集モード
- GET `/api/teachers/:id` で既存データ取得
- パスワードフィールド非表示
- PUT `/api/teachers/:id` で更新
- 成功時 → 講師詳細ページへ遷移

### フォームフィールド一覧

| フィールド | 型 | 必須 | 新規時のみ | 選択肢 |
|-----------|-----|------|-----------|--------|
| 氏名 | text | 必須 | | |
| メールアドレス | email | 必須 | | |
| パスワード | password | 必須 | 新規時のみ表示 | |
| 電話番号 | text | 必須 | | |
| 雇用形態 | select | 必須 | | 常勤(full_time) / 非常勤(part_time) |
| ステータス | select | 必須 | | 稼働中(active) / 非稼働(inactive) |
| 担当可能科目 | トグルボタン群 | 任意 | | 下記科目一覧参照 |

### 担当可能科目（トグルボタン）
選択可能な科目一覧。タップで選択/解除を切り替え。

| 科目 |
|------|
| 数学、英語、国語、物理、化学、生物、日本史、世界史、地理 |

### デザイン仕様

#### 科目トグルボタン
- `flex flex-wrap gap-2` で横並び配置
- 未選択時: `bg-gray-100 text-gray-700`、ホバーで `bg-gray-200`
- 選択時: `bg-blue-600 text-white`
- 形状: `px-3 py-1 rounded-full text-sm`

#### その他
- レイアウト・入力欄スタイル・ボタンは生徒編集ページと同一パターン
