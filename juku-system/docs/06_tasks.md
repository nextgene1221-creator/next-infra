# タスク管理 詳細仕様書

## 基本情報

| ページ | パス | ソースファイル | レンダリング | アクセス可能ロール |
|--------|------|---------------|-------------|-------------------|
| タスク一覧 | `/tasks` | `src/app/(authenticated)/tasks/page.tsx` | サーバーサイド | admin, teacher, student |
| タスク作成・編集 | `/tasks/:id/edit` | `src/app/(authenticated)/tasks/[id]/edit/page.tsx` | クライアントサイド | admin, teacher |

### API エンドポイント

| メソッド | パス | 説明 | 権限 |
|---------|------|------|------|
| GET | `/api/tasks/:id` | タスク情報取得 | 認証済み全ロール |
| POST | `/api/tasks` | タスク新規作成 | admin, teacher |
| PUT | `/api/tasks/:id` | タスク更新 | admin, teacher |

## 1. タスク一覧ページ (`/tasks`)

### 機能説明

#### 検索・フィルタ機能

| パラメータ | 説明 | 選択肢 |
|-----------|------|--------|
| `status` | ステータスフィルタ | 全ステータス / 未着手 / 進行中 / 完了 / 期限超過 |
| `subject` | 科目フィルタ | 全科目 / 数学 / 英語 / 国語 / 物理 / 化学 / 生物 / 日本史 / 世界史 / 地理 |

#### ロール別データスコープ
- **admin**: 全タスクを表示
- **teacher**: 自身が作成したタスクのみ表示（`teacher_id` でフィルタ）
- **student**: 自身に割り当てられたタスクのみ表示（`student_id` でフィルタ）

#### 表示件数・ソート
- 最大50件、期限日昇順（`dueDate ASC`）で表示

#### 新規作成ボタン
- admin, teacher ロールの場合のみ「新規作成」ボタンを表示
- 遷移先: `/tasks/new/edit`

### デザイン仕様

#### テーブルカラム

| カラム | 表示内容 | スタイル |
|--------|---------|---------|
| タスク名 | `title` | `font-medium text-gray-900` |
| 科目 | `subject` | テキスト |
| 生徒 | `student.user.name` | テキスト |
| 期限 | `dueDate`（ja-JP形式） | テキスト |
| ステータス | バッジ表示 | 色分けバッジ |
| 操作 | 「編集」リンク | admin, teacher のみ表示 |

#### ステータスバッジの色分け

| ステータス | 表示名 | バッジ色 |
|-----------|--------|---------|
| pending | 未着手 | 黄（`bg-yellow-100 text-yellow-800`） |
| in_progress | 進行中 | 青（`bg-blue-100 text-blue-800`） |
| completed | 完了 | 緑（`bg-green-100 text-green-800`） |
| overdue | 期限超過 | 赤（`bg-red-100 text-red-800`） |

#### 操作カラム
- admin, teacher: 「編集」リンク（青色テキスト → `/tasks/:id/edit`）
- student: 操作カラムは空

#### 0件時表示
- 「タスクがありません」を中央表示

## 2. タスク作成・編集ページ (`/tasks/:id/edit`)

### 機能説明

#### 新規作成モード（`id = "new"`）
- ステータス選択フィールドは非表示（自動的に `pending` で作成）
- POST `/api/tasks` で新規作成
- 作成者（teacher_id）は現在ログイン中の講師が自動設定される
- 成功時 → タスク一覧ページへ遷移

#### 編集モード（`id ≠ "new"`）
- GET `/api/tasks/:id` で既存データ取得
- ステータス選択フィールドが表示される
- PUT `/api/tasks/:id` で更新
- 成功時 → タスク一覧ページへ遷移

#### データ取得
- ページ読み込み時に GET `/api/students-list` で担当生徒リストを非同期取得

### フォームフィールド一覧

| フィールド | 型 | 必須 | 条件 | 初期値 | 選択肢 |
|-----------|-----|------|------|--------|--------|
| タスク名 | text | 必須 | | 空 | |
| 生徒 | select | 必須 | | 未選択 | APIから取得した生徒リスト |
| 科目 | select | 必須 | | 未選択 | 9科目 |
| 期限 | date | 必須 | | 今日の日付 | |
| ステータス | select | 必須 | 編集時のみ表示 | pending | 未着手/進行中/完了/期限超過 |
| 内容・詳細 | textarea（4行） | 任意 | | 空 | |

### デザイン仕様

#### レイアウト
- 最大幅 `max-w-2xl` の白背景カード（`p-6 space-y-4`）
- フォーム項目: 2カラムグリッド（`md:grid-cols-2 gap-4`）
- 内容・詳細テキストエリア: 全幅

#### ボタン
- 「保存」ボタン: 青色背景、保存中は「保存中...」＋非活性
- 「キャンセル」ボタン: グレー背景、ブラウザ戻る
