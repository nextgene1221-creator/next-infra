# 学習進捗管理 詳細仕様書

## 基本情報

| ページ | パス | ソースファイル | レンダリング | アクセス可能ロール |
|--------|------|---------------|-------------|-------------------|
| 進捗一覧 | `/progress` | `src/app/(authenticated)/progress/page.tsx` | サーバーサイド | admin, teacher, student |
| 進捗入力 | `/progress/new` | `src/app/(authenticated)/progress/new/page.tsx` | クライアントサイド | admin, teacher |
| 進捗詳細 | `/progress/:id` | `src/app/(authenticated)/progress/[id]/page.tsx` | サーバーサイド | admin, teacher, student |

### API エンドポイント

| メソッド | パス | 説明 | 権限 |
|---------|------|------|------|
| POST | `/api/progress` | 進捗記録作成 | admin, teacher |
| GET | `/api/students-list` | 生徒選択リスト取得 | admin, teacher |

## 1. 進捗一覧ページ (`/progress`)

### 機能説明

#### 検索・フィルタ機能

| パラメータ | 説明 | 表示条件 | 選択肢 |
|-----------|------|---------|--------|
| `student` | 生徒フィルタ | admin, teacher のみ | 全生徒 / 個別生徒名 |
| `subject` | 科目フィルタ | 全ロール | 全科目 / 各科目名 |

#### ロール別データスコープ
- **admin**: 全進捗記録を表示
- **teacher**: 自身が記録した進捗のみ表示（`teacher_id` でフィルタ）
- **student**: 自身の進捗のみ表示（`student_id` でフィルタ）、生徒選択ドロップダウンは非表示

#### 表示件数
- 最新50件（日付降順）

#### 進捗入力ボタン
- admin, teacher ロールの場合のみ「進捗入力」ボタンを表示
- 遷移先: `/progress/new`

### デザイン仕様

#### テーブルカラム

| カラム | 表示内容 | スタイル |
|--------|---------|---------|
| 日付 | `date`（ja-JP形式） | 青色リンク → 進捗詳細ページ |
| 生徒 | `student.user.name` | テキスト |
| 科目 | `subject` | テキスト |
| 単元 | `topic` | テキスト |
| 理解度 | `understandingLevel`/5 | 色分けバッジ |
| 講師 | `teacher.user.name` | テキスト |

#### 理解度バッジの色分け

| 理解度 | バッジ色 |
|--------|---------|
| 4〜5 | 緑（`bg-green-100 text-green-800`） |
| 3 | 黄（`bg-yellow-100 text-yellow-800`） |
| 1〜2 | 赤（`bg-red-100 text-red-800`） |

#### 0件時表示
- 「進捗記録がありません」を中央表示

## 2. 進捗入力ページ (`/progress/new`)

### 機能説明

授業後に講師が学習進捗を記録するフォーム。

#### データ取得
- ページ読み込み時に GET `/api/students-list` で担当生徒リストを非同期取得
- 講師ロールの場合、担当生徒のみがリストに表示される

#### 送信処理
- POST `/api/progress` で進捗記録を作成
- 記録者（teacher_id）は現在ログイン中の講師が自動設定される
- 成功時 → 作成された進捗記録の詳細ページへ遷移

### フォームフィールド一覧

| フィールド | 型 | 必須 | 初期値 | 選択肢 |
|-----------|-----|------|--------|--------|
| 生徒 | select | 必須 | 未選択 | APIから取得した生徒リスト |
| 科目 | select | 必須 | 未選択 | 数学/英語/国語/物理/化学/生物/日本史/世界史/地理 |
| 授業日 | date | 必須 | 今日の日付 | |
| 学習内容・単元 | text | 必須 | | |
| 理解度 | range（1〜5） | 必須 | 3 | スライダー |
| コメント | textarea（3行） | 任意 | | |

### デザイン仕様

#### レイアウト
- 最大幅 `max-w-2xl` の白背景カード
- 2カラムグリッド（`md:grid-cols-2`）

#### 理解度スライダー
- `type="range"` によるネイティブスライダー
- ラベルに現在値を「理解度 (n/5)」形式でリアルタイム表示
- スライダー下部に1〜5の目盛りラベル（`text-xs text-gray-400`）を等間隔で配置

#### ボタン
- 「保存」ボタン: 青色、保存中は「保存中...」＋非活性
- 「キャンセル」ボタン: グレー、ブラウザ戻る

## 3. 進捗詳細ページ (`/progress/:id`)

### 機能説明

進捗記録の全情報を表示する読み取り専用ページ。

#### 表示項目

| 項目 | 表示内容 |
|------|---------|
| 生徒 | 生徒名（青色リンク → 生徒詳細ページ） |
| 講師 | 講師名 |
| 科目 | 科目名 |
| 授業日 | 日付（ja-JP形式） |
| 単元 | 学習内容名 |
| 理解度 | n/5（色分けバッジ） |
| コメント | テキスト（改行保持: `whitespace-pre-wrap`）。コメントがない場合は非表示 |

#### ナビゲーション
- ヘッダー右側に「一覧に戻る」リンク → `/progress`

### デザイン仕様

- 最大幅 `max-w-2xl` の白背景カード
- 定義リスト形式（ラベル幅 `w-32`）
- 項目間: `space-y-4`
