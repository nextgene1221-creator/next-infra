## 機能説明

### 認証機能
- メールアドレスとパスワードによるCredentials認証（NextAuth.js）
- 認証成功時、`/dashboard` へ自動遷移
- 認証失敗時、画面上部にエラーメッセージ「メールアドレスまたはパスワードが正しくありません」を表示
- 送信中はボタンを非活性化し「ログイン中...」テキストを表示

### 認証フロー
1. ユーザーがメールアドレス・パスワードを入力
2. 「ログイン」ボタンを押下
3. NextAuth `signIn("credentials")` をリダイレクトなし（`redirect: false`）で実行
4. 成功 → `router.push("/dashboard")`
5. 失敗 → エラーメッセージ表示、ローディング解除

### セッション管理
- JWT方式でセッションを管理
- セッションにはユーザーID、名前、メール、ロール（admin/teacher/student）を格納
- 未認証ユーザーが認証必要ページにアクセスした場合、`/login` にリダイレクト

## デザイン仕様

### レイアウト
- 画面全体を灰色背景（`bg-gray-50`）で覆い、中央に白いログインカードを配置
- ログインカードは最大幅 `max-w-md`（28rem / 448px）
- カードには角丸（`rounded-lg`）とシャドウ（`shadow`）を適用
- カード内はパディング `p-8`（2rem）

### ヘッダー部分
- タイトル「塾管理システム」：`text-3xl`、太字、中央寄せ、色 `text-gray-900`
- サブタイトル「アカウントにログイン」：`text-sm`、中央寄せ、色 `text-gray-600`、上余白 `mt-2`

### フォーム部分
- 各入力フィールド間の間隔：`space-y-4`
- **メールアドレス入力**
  - ラベル：「メールアドレス」（`text-sm font-medium text-gray-700`）
  - 入力欄：`type="email"`、`required`
  - 枠線 `border-gray-300`、角丸 `rounded-md`、シャドウ `shadow-sm`
  - フォーカス時：青色リング（`ring-blue-500`）と青色枠線（`border-blue-500`）
- **パスワード入力**
  - ラベル：「パスワード」
  - 入力欄：`type="password"`、`required`
  - スタイルはメールアドレスと同一

### ボタン
- 全幅（`w-full`）、青色背景（`bg-blue-600`）、白文字
- ホバー時：`bg-blue-700`
- フォーカス時：青色リング（`ring-2 ring-blue-500 ring-offset-2`）
- 非活性時：不透明度50%（`disabled:opacity-50`）

### エラー表示
- 赤色の薄い背景（`bg-red-50`）、赤色枠線（`border-red-200`）、赤色テキスト（`text-red-700`）
- フォーム上部に表示される
- 角丸（`rounded`）、パディング `px-4 py-3`
