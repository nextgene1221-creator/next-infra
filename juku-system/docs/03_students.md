# 生徒管理 詳細仕様書

## 基本情報

| ページ | パス | ソースファイル | レンダリング | アクセス可能ロール |
|--------|------|---------------|-------------|-------------------|
| 生徒一覧 | `/students` | `src/app/(authenticated)/students/page.tsx` | サーバーサイド | admin, teacher |
| 生徒詳細 | `/students/:id` | `src/app/(authenticated)/students/[id]/page.tsx` | サーバーサイド | admin, teacher |
| 生徒登録・編集 | `/students/:id/edit` | `src/app/(authenticated)/students/[id]/edit/page.tsx` | クライアントサイド | admin |

### API エンドポイント

| メソッド | パス | 説明 | 権限 |
|---------|------|------|------|
| GET | `/api/students/:id` | 生徒情報取得 | 認証済み全ロール |
| POST | `/api/students` | 生徒新規登録 | admin のみ |
| PUT | `/api/students/:id` | 生徒情報更新 | admin のみ |

## 1. 生徒一覧ページ (`/students`)

### 機能説明

#### 検索・フィルタ機能
URLクエリパラメータによるサーバーサイドフィルタリング。

| パラメータ | 説明 | 選択肢 |
|-----------|------|--------|
| `q` | 名前での部分一致検索 | 自由入力テキスト |
| `status` | ステータスフィルタ | 全ステータス / 在籍(active) / 休塾(inactive) / 退塾(withdrawn) |
| `grade` | 学年フィルタ | 全学年 / 高1(1) / 高2(2) / 高3(3) |

#### ロール別データスコープ
- **admin**: 全生徒を表示
- **teacher**: 自身が担当する生徒のみ表示（`student_teacher` テーブルで紐付く生徒）

#### 新規登録ボタン
- admin ロールの場合のみ、ヘッダー右側に「新規登録」ボタンを表示
- 遷移先: `/students/new/edit`

### デザイン仕様

#### ヘッダー
- タイトル「生徒一覧」（`text-2xl font-bold`）とボタンを `flex justify-between` で左右配置
- 「新規登録」ボタン: 青色背景（`bg-blue-600`）、白文字、角丸 `rounded-md`

#### 検索フォーム
- 白背景カード（`bg-white p-4 rounded-lg shadow`）、下余白 `mb-6`
- フォーム要素を `flex gap-4 flex-wrap` で横並び配置
- 検索テキスト入力: `border-gray-300 rounded-md px-3 py-2 text-sm`
- セレクトボックス: テキスト入力と同一スタイル
- 「検索」ボタン: 黒系背景（`bg-gray-800`）、白文字

#### テーブル
- 白背景カード（`bg-white rounded-lg shadow overflow-hidden`）
- テーブルヘッダー: `bg-gray-50`、カラム名は `text-xs font-medium text-gray-500 uppercase`
- 行ホバー: `hover:bg-gray-50`
- セル: パディング `px-6 py-4`、テキスト `text-sm`

#### テーブルカラム

| カラム | 表示内容 | スタイル |
|--------|---------|---------|
| 名前 | `user.name` | 青色リンク（`text-blue-600 hover:underline`）→ 詳細ページ |
| 学年 | 「高{grade}」 | テキスト |
| 高校名 | `schoolName` | テキスト |
| ステータス | バッジ表示 | 色分けバッジ（下記参照） |
| 入塾日 | `enrollmentDate`（ja-JP形式） | テキスト |

#### ステータスバッジの色分け

| ステータス | 表示名 | バッジ色 |
|-----------|--------|---------|
| active | 在籍 | 緑（`bg-green-100 text-green-800`） |
| inactive | 休塾 | 黄（`bg-yellow-100 text-yellow-800`） |
| withdrawn | 退塾 | 赤（`bg-red-100 text-red-800`） |

#### 0件時表示
- テーブル内にコールスパン5の中央寄せメッセージ「生徒が見つかりません」

## 2. 生徒詳細ページ (`/students/:id`)

### 機能説明

生徒の全情報を4つのセクションに分けて表示する。

#### セクション構成

| セクション | 表示内容 |
|-----------|---------|
| 基本情報 | メール、学年、高校名、ステータス、入塾日、保護者情報、備考 |
| 担当講師 | 紐付いている講師名と担当科目（各講師名は講師詳細へのリンク） |
| 最近の学習進捗 | 直近10件の進捗記録（科目、単元、理解度、日付、講師名） |
| 最近のタスク | 直近10件のタスク（タイトル、科目、期限、ステータスバッジ） |

#### 編集ボタン
- ヘッダー右側に「編集」ボタン → `/students/:id/edit` へ遷移

#### 存在しないIDの場合
- Next.js `notFound()` を呼び出し404ページを表示

### デザイン仕様

#### レイアウト
- 2カラムグリッド（`grid-cols-1 lg:grid-cols-2 gap-6`）
- 各セクションは白背景カード（`bg-white rounded-lg shadow p-6`）

#### 基本情報セクション
- 定義リスト（`<dl>`）形式
- ラベル幅: `w-32`（8rem）、色 `text-sm text-gray-500`
- 値: `text-sm text-gray-900`
- 項目間: `space-y-3`

#### 担当講師セクション
- リスト形式、各項目は `bg-gray-50 rounded p-2`
- 講師名: 青色リンク → 講師詳細ページ
- 科目: 右寄せ（`text-xs text-gray-500`）

#### 学習進捗セクション
- リスト形式、各項目は `bg-gray-50 rounded p-2`
- 行クリックで進捗詳細ページへ遷移
- 上段: 科目 - 単元（左） / 理解度 x/5（右）
- 下段: 日付 / 講師名

#### タスクセクション
- リスト形式、左にタスク名・科目・期限、右にステータスバッジ

## 3. 生徒登録・編集ページ (`/students/:id/edit`)

### 機能説明

#### 新規登録モード（`id = "new"`）
- 全フィールドが空の状態で表示
- パスワード入力フィールドが表示される（必須）
- POST `/api/students` で新規作成
- 成功時、作成された生徒の詳細ページへ遷移

#### 編集モード（`id ≠ "new"`）
- GET `/api/students/:id` で既存データを取得しフォームにプリセット
- パスワードフィールドは非表示
- PUT `/api/students/:id` で更新
- 成功時、生徒詳細ページへ遷移

#### エラーハンドリング
- APIエラー時、フォーム上部に赤色エラーメッセージを表示
- メールアドレス重複時: 「このメールアドレスは既に使用されています」

### デザイン仕様

#### レイアウト
- 最大幅 `max-w-2xl`（42rem / 672px）の白背景カード
- カード内: `p-6 space-y-4`

#### フォーム項目
- 2カラムグリッド（`grid-cols-1 md:grid-cols-2 gap-4`）
- ラベル: `text-sm font-medium text-gray-700`
- 入力欄: `border-gray-300 rounded-md px-3 py-2 text-sm`、上余白 `mt-1`

#### フォームフィールド一覧

| フィールド | 型 | 必須 | 新規時のみ | 選択肢 |
|-----------|-----|------|-----------|--------|
| 氏名 | text | 必須 | | |
| メールアドレス | email | 必須 | | |
| パスワード | password | 必須 | 新規時のみ表示 | |
| 学年 | select | 必須 | | 高1 / 高2 / 高3 |
| 高校名 | text | 必須 | | |
| ステータス | select | 必須 | | 在籍 / 休塾 / 退塾 |
| 入塾日 | date | 必須 | | |
| 保護者氏名 | text | 必須 | | |
| 保護者電話番号 | text | 必須 | | |
| 保護者メール | email | 必須 | | |
| 備考 | textarea（3行） | 任意 | | |

#### ボタン
- 「保存」ボタン: 青色背景（`bg-blue-600`）、保存中は「保存中...」表示＋非活性
- 「キャンセル」ボタン: グレー背景（`bg-gray-200`）、ブラウザ戻る（`router.back()`）
